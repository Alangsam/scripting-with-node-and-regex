JavaScript Functions: Regex

functions documented

wheresWaldo - if given a string, will tell you if "Waldo" is found inside

matchVowels - will find all the vowels in a string

matchNotVowels - everything in the string that's not a number or vowel

moreThanOnce - shows you everytime there's more than one s

findOneOrMore - finds al the c's in a string

beginningOf - see if a string begins with "The"

endingOf - see if string ends with "end"

matchAlphaNums - show every alphanumeric element in a string

matchNonAlphaNums - show every nonalphanumeric element

matchNums - find all the numbers in a string

matchNonNums - find all everything thats not a number

matchNoWhite - find everything thats not whitespace

posNegLookahead - cant start with a number, has to be 5 characters long, and has to end with at least 2 numbers

mixedGrouping - can find either Eleanor or Franklin Roosevelt

captureGroupsReplace - replaces the first three words, with themselves but in reverse order, dont use punctuation

iHateWhiteSpace - remove white space from the front and end of a string
